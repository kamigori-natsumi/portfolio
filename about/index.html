ABOUT

  • 名前

    上郡　菜摘 かみごおり　なつみ

  • MBTI(16パーソナリティー)

    提唱者タイプ

  • 経歴

    1994年12月15日生まれの射手座。兵庫県西脇生まれの西脇育ち。京都光華女子大学健康栄養学科卒業。管理栄養士として6年働きました。

  • スキル

    • Illustrator　基本的な操作ができます。ベジェ曲線をスムーズに扱え、ロゴやイラスト等の作成ができます。トンボをつけた入稿のデータが作れます。
    • Photoshop　解像度やレイヤー、マスクの仕組みを理解しており、画像の色補正、合成、編集がでます。画像の保存形式jpg、png、svg等の使い分けができます。
    • Figma　ワイヤーフレーム、デザインカンプの作成ができます。またオートレイアウトを使用してのバナー作成ができます。このポートフォリオもFigmaで作成しました。
    • Visual Stadio code　HTML、CSS、jacascript、jQueryが使用できます。javascriptは基本的なことができるレベルなので、自由に使用できるよう今後も力を入れて勉強していきます。

  • 趣味

    小学生の頃からイラストを描くことが大好きです。普段は版権のイラストを描いています。今は原神というゲームにハマっているため、その作品のイラストを描くことが多いです。

    また3年前からバイオリンを習い始めました。最初は全然弾けませんでしたが少しずつ音色や技術が上がり弾ける曲が増えていくことに達成感と喜びを感じます。

    他にハンドメイドやカメラ、カラオケも好きです。
